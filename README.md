# movie-app

SPA веб-приложение в котором можно найти фильмы по запросу и сохранить в личном кабинете.

#СТЕК:
* Vuejs(options api)
* Vuex
* Vue Router
* API Фильмов: https://api.kinopoisk.dev
* Firebase
* SCSS
* Адаптивная верстка
